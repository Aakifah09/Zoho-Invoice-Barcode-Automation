## Project Overview
This project automates invoice data retrieval from Zoho Books and generates barcode stickers using Zoho Creator.

Invoice data is fetched using APIs and workflows, parsed, and used to produce ready-to-print barcode labels matching client specifications.

---

## Key Features
- Automated invoice data retrieval
- API integration with Zoho Books
- Barcode generation from MPN
- Sticker template design (50mm × 25mm)
- Workflow automation using Deluge scripting
- Print-ready sticker generation

---

## Technologies Used
- Zoho Creator
- Zoho Books API
- Deluge scripting
- HTML/CSS templates
- BWIP-JS Barcode API
- Label printer configuration
